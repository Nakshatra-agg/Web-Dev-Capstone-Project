# Web-Dev Capstone Project

## Theme
A static, responsive web development project demonstrating layout, styling, and use of assets (images) built using **HTML and CSS**.

## Sections Implemented
- **Header / Navigation**: Site logo and menu links.  
- **Hero Section**: Landing image/banner with prominent call-to-action.  
- **About Section**: Introduction of the project or site purpose.  
- **Gallery / Images**: Showcasing images used in the site.  
- **Footer**: Contact information and links.  

## Layout Techniques & Breakpoints
- **Layout Techniques**:  
  - Flexbox for navigation and sections alignment.  
  - CSS Grid for image gallery layout.  
  - Margins, padding, and typography for spacing and readability.  
- **Breakpoints**:  
  - Mobile: up to 480px  
  - Tablet: 481px–768px  
  - Desktop: 769px and above  
- Media queries used to adjust layout and font sizes for different screen widths.
